About Smart Contact Manager

A smarter way to manage your contacts — with cloud-powered image storage, real-time sync, and seamless accessibility across devices.

🌍 Our Vision

To build a clean, simple, and secure cloud-based platform that replaces outdated contact lists with something modern and smart.

👥 Built for Everyone

Whether you're a freelancer, a student, or a team managing hundreds of clients — SCM keeps you organized, fast.

☁️ Cloud + Image Smartness

Every contact is more than just a name — attach a profile picture, sync it to the cloud (via Cloudinary), and access it anytime, anywhere.

🛠 Tech Stack

Java + Spring Boot + Thymeleaf + Tailwind CSS + Cloudinary. We’ve built SCM with modern tech to deliver performance and security.

🔐 Built with Privacy in Mind

Your data belongs to you. With secure logins (OAuth & JWT), encrypted storage, and private image handling, SCM puts your privacy first.
